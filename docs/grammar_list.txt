1. Arithmetic and Elementary Algebra

= + — Addition
= - — Subtraction
= \pm — Plus-minus
= \times — Multiplication (Cross)
= \cdot — Multiplication (Dot)
= \div — Division
= / — Slash division
= = — Equality
= \neq — Inequality
= < — Strictly less than
= > — Strictly greater than
= \le — Less than or equal
= \ge — Greater than or equal
= \approx — Approximation
= \sim — Proportionality / Tilde

= |x| — Absolute value / Modulo

2. Analysis and Calculus
f(x) — Function:
= f' — Lagrange's notation derivative
= \frac{df}{dx} — Leibniz's notation derivative
= \frac{\partial f}{\partial x} — Partial differentiation
= \dot{x} — Newton's notation (time) derivative
= \nabla f — Gradient operation
= \Delta f — Laplacian operation
= \square f — D'Alembertian operation
= (f \circ g) — Composition
= (f \ast g) — Convolution
\lim — Limit:
= \lim_{x \to a} — Standard limit
= \limsup — Limit superior
= \liminf — Limit inferior
\int — Integral:
= \int_{a}^{b} — Definite integration
= \oint — Contour integration
= \iint — Double integration
= \iiint — Triple integration
= \int \dots dx — Indefinite integration
\sum — Summation:
= \sum_{i=1}^{n} — Finite summation
= \sum_{i=0}^{\infty} — Infinite series summation
\prod — Product:
= \prod_{i=1}^{n} — Finite product
= \coprod — Coproduct operation
