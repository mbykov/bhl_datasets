./run_safe.sh 100 3 50 14000 0.2

⚙️ Оптимальные настройки для RTX 5060 Ti (16GB)
Режим	Потоки	Батч	Порог MB	Задержка	Скорость	Стабильность
Стабильный	1	25	12000	0.5	1x	⭐⭐⭐⭐⭐
Рекомендуемый	2	30	14000	0.3	1.3x	⭐⭐⭐⭐
Максимальный	3	50	15000	0.2	1.5x	⭐⭐⭐

Рекомендую начать с режима 2 потока - это лучший баланс скорости и стабильности для вашей карты.
