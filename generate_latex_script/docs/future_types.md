Возможные пропущенные важные типы:
1. Унарные операции с вложенностью
text

sym(sym@)  - например: sin(sin x)
next(next@) - например: ∫(∫ x dx)

2. Тернарные и многоместные операции
text

sym(@, @) - например: \frac{a}{b}
sym(@, @, @) - например: \sqrt[3]{x}

3. Цепочки операций (3+ операндов)
text

sym@ ? next@ ? sym@ - например: a + b - c
sym@ ? next@ ? next@ - например: a + b + c

4. Вложенность с обеих сторон от оператора
text

(sym@) ? (next@) - со скобками
sym(next@) ? sym@ - вложенность только слева
sym@ ? next(sym@) - вложенность только справа

5. Двойная вложенность
text

sym(next(sym@)) - тройная вложенность
sym(sym(next@))

6. Операции с индексами/степенями
text

sym@^{@} - степень
sym@_{@} - индекс
sym@^{@}_{@} - степень и индекс

7. Специальные конструкции из вашего исходного scheme

    \frac{@}{@} - дробь

    \sum_{@}^{@} @ - сумма с пределами

    \int_{@}^{@} @ d@ - интеграл с пределами

Мои рекомендации по приоритету:

Критически важные (стоит добавить):

    Дробные конструкции (\frac{}{}) - очень частотны в математике

    Степени и индексы - основа основ

    Цепочки из 3 операндов - для обучения пониманию приоритетов

Важные (хорошо бы добавить):

    Двойная вложенность - для глубокого понимания

    Унарные операции с одним символом (sym(sym@))

Менее критичные (можно позже):

    Скобки вокруг выражений
